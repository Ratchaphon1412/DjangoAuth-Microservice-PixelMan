- Python 3.11
- Django

- [Python HVAC HASICORP VAULT](https://hvac.readthedocs.io/en/stable/overview.html)
- [Vault](https://www.vaultproject.io/)

- [Django Environment](https://django-environ.readthedocs.io/en/latest/)
